# E-Commerce backend
backend for [E-commerce frontend](https://github.com/real0ta/e-commerce_frontend)

## Running locally
1. Clone this respository
2. run 
```bash 
npm install
```
3. Create .env file in config directory and add the following environment variable
`JWT_KEY `

`JWT_REFRESH_KEY`

`PORT`

`STRIPE_KEY`

`URL` mongodb url
5. run 
```bash 
npm run dev
```
